\noindent \textbf{Solução:}
Podemos criar uma representação vetorial de tamanho $n$ em que cada entrada $i$ desse vetor corresponde à máquina para a qual a tarefa $i$ foi alocada. Por exemplo, se temos $n = 3$ tarefas e $m = 2$ máquinas, uma possível representação seria \fbox{0} \fbox{0} \fbox{1}, indicando que as duas primeiras tarefas serão alocadas à máquina 0, e a última, à máquina 1.
